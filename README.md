# Smart Retail Insights: Data-Driven Decision Support System

## Project Overview
This project analyzes retail data to generate insights and support business decision making using data science techniques.

## Features
- Data preprocessing
- Exploratory data analysis
- Machine learning model
- Demand prediction
- Interactive dashboard

## Technologies Used
- Python
- Pandas
- Scikit-learn
- SQL
- Power BI / Streamlit

## How to Run
1. Install requirements
pip install -r requirements.txt

2. Run the application
python app.py
